Skills and Knowledge

Throughout my placement, I had the opportunity to delve deep into cybersecurity concepts and practices. A significant portion of my placement involved working with Splunk, a powerful Security Information and Event Management (SIEM) solution. I honed my skills in navigating Splunk dashboards, performing advanced searches using the Splunk Search Processing Language (SPL), and creating custom queries. This experience not only expanded my technical skill set but also introduced me to a widely used cybersecurity tool that is invaluable for threat detection and analysis.
